CodeGPT

Your coding companion!

CodeGPT is a ChatGPT clone webapp that I developed using ReactJS, JavaScript, HTML, and CSS. It is a useful coding companion that can help you learn about the many different aspects of computer science.

I implemented OpenAI's "text-davinci-003" API algorithm so that the chat bot can recognize and respond to text, with a focus on code and computer science questions.

I implemented the server for this webapp in JavaScript and hosted it through Vercel and Render.

It is very powerful as an on-the-fly learning companion and research tool, go ahead and give it a try! Click here to visit CodeGPT.
