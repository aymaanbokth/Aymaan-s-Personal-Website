DBANK Crypto

Custom crypto token

This was a project I started so that I could learn more about Web3.0, the blockchain, and cryptocurrencies.

I learned a new decentralized finance programming language called Motoko, and also learned about the Internet Computer Protocol. Both of these new tools allowed me to develop a full stack decentralized finance application where users can claim, send, and receive my own custom cryptcurrency tokens. I used ReactJS, CSS, and HTML to build the front end, and used Motoko to create the back end.

This project taught me a lot of what I know about decentralized application development, and the many use cases and benefits that it has. I hope to make a couple of other projects that build on this foundational knowledge.
